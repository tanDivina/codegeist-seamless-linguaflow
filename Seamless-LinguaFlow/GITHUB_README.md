## ğŸš€ Quick Start

### **Prerequisites**
```bash
# Install Forge CLI
npm install -g @forge/cli

# Login to Forge
forge login

# Set Gemini API key
forge variables set --encrypt GEMINI_API_KEY your_key_here
```

### **Deployment**
```bash
# Clone and deploy
git clone https://github.com/your-username/seamless-linguaflow
cd seamless-linguaflow/Seamless-LinguaFlow
forge deploy
forge install --product jira
```

### **Verification**
1. Open any Jira issue
2. Look for "Seamless LinguaFlow" panel
3. Test translation and consistency features
4. Try Rovo agent with "Check consistency of this translation"
